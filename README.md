**Healthcare Patient Analysis System (Synthetic Data)**

 Project Overview This project focuses on analyzing healthcare patient data using synthetic datasets. It simulates real-world patient records and performs Exploratory Data Analysis (EDA) to identify patterns, relationships, and high-risk patients.

 **Objectives**

Generate synthetic patient data (10,000 records) Perform data cleaning and preprocessing

**Analyze health parameters:**

.Blood Pressure .Sugar Level .Cholesterol .Heart Rate Classify patients into risk categories (Low, Medium, High) Identify high-risk patients Perform group-based and relationship analysis Visualize data using charts and graphs Derive meaningful healthcare insights

 Dataset Description

**Column Name Description**

Patient_ID Unique ID Age 20–80 Gender Male / Female Blood_Pressure 80–180 Sugar_Level 70–200 Cholesterol 150–300 Heart_Rate 60–120

Technologies Used

**Python  Pandas NumPy Matplotlib Seaborn
**
** Project Workflow**

    

Data Generation Created synthetic dataset using NumPy and Pandas

Data Cleaning & Preprocessing Handled missing values Checked data types Ensured data consistency

Exploratory Data Analysis (EDA) Mean, Median, Mode, Standard Deviation Distribution analysis

Risk Classification Categorized patients: Low Risk Medium Risk High Risk

High-Risk Identification Filtered patients needing immediate attention

Group-Based Analysis Age group vs health condition Gender-wise comparison Risk distribution

Relationship Analysis Blood Pressure vs Age Cholesterol vs Heart Rate Sugar Level vs Risk Category

Data Visualization Bar Charts Pie Charts Histograms Scatter Plots Box Plots Heatmaps

 Key Insights Identified high-risk patient percentage Observed correlation between health parameters Detected patterns across age groups and gender

 Outcomes Early detection of health risks Improved patient monitoring Supports healthcare decision-making

** Conclusion**

This project demonstrates how synthetic healthcare data can be effectively analyzed using Python to gain meaningful insights and support medical decisions.
